Mobile-specific improvements

To make the mobile app even stronger than the website, consider adding:

A blue accent (#3B82F6) for informational actions (maps, GPS, help, syncing), reducing the visual dominance of green while maintaining the brand identity.

Color-coded modules: Green for Safety, Purple for Reports, Blue for Location/Assets, Amber for Alerts, Red for Critical Incidents. This improves recognition speed for field workers.

A dark mode using near-black surfaces (#0F172A) with the same accent colors, useful for low-light environments and battery savings on OLED devices.

This design system provides a scalable foundation for building a cohesive HSE mobile application in Figma or implementing it with Flutter using Material 3 while preserving the visual identity shown in the website.
